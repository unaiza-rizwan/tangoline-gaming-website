# Tangoline Inc. — Frontend Project

A gaming e-commerce website for Pakistan's premier gaming store, selling PS5, Xbox, Nintendo consoles and games.

## 🎮 Live Preview

Open `index.html` in your browser to get started.

## 📁 Project Structure

```
tangoline_Inc/
├── index.html            # Homepage
├── product.html          # Products category page
├── console.html          # Consoles listing (PS5, Xbox, Nintendo)
├── games.html            # Games listing
├── about-us.html         # About page
├── contact-us.html       # Contact form
├── login.html            # Login page
├── sign-in.html          # Sign-up page
├── shared.css            # Shared stylesheet (all pages use this)
├── style.css             # Login/signup animated background
├── signupstyle.css       # Sign-up page styles
└── console and games/    # Product detail .docx files
```

## 🛠️ Tech Stack

- **HTML5** — Semantic, accessible markup
- **CSS3** — Custom properties, animations, responsive grid
- **Bootstrap 5** — Responsive layout utilities
- **Google Fonts** — Orbitron (headings) + Rajdhani (body)

## ✨ Features

- Sticky responsive navbar with active page highlighting
- Animated hero section with glowing text effect
- Product grid with hover effects and smooth transitions
- Brand navigation (PlayStation / Xbox / Nintendo filters)
- Working contact form
- Newsletter subscription form
- Fully responsive (mobile, tablet, desktop)
- Consistent dark gaming theme across all pages

## 🚀 Getting Started

No build tools needed — it's pure HTML/CSS/JS.

```bash
git clone https://github.com/YOUR_USERNAME/tangoline-inc.git
cd tangoline-inc
# Open tagoline.Inc.html in your browser
```

Or use VS Code Live Server for hot reload.

## 📦 Deployment

Works on any static hosting — GitHub Pages, Netlify, Vercel:

**GitHub Pages:**
1. Push to GitHub
2. Settings → Pages → Source: main branch / root
3. Your site goes live at `https://username.github.io/tangoline-inc/tagoline.Inc.html`
